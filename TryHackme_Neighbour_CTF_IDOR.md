By Rully Miftahur Rozaq

Langkah Langkah

Untuk Langkah pertama saya mengaktifkan machine di tryhackme agar mendapat IP nya

Setelah mendapatkan IP nya saya mengakses IP yang tersedia di kasus saya IP nya Adalah 10.48.173.31





Ini Adalah tampilan website nya disini terdapat form login 

Dikarenakan saya tidak punya akun saya login dengan akun guest dengan memencet tombol CTRL + U di keyboard dan muncul tampilan source code nya









Kemudian saya memasukkan username nya guest dan password nya Adalah guest 



Dan ini Adalah tampilan website nya setelah saya login menggunakan akun guest 

































Setelah saya berhasil login saya mencoba melakukan IDOR (Insecure Direct Object Reference) sebuah kerentanan Dimana pengguna memanipulasi ID untuk mengakses, mengubah, atau menghapus data milik orang lain. Disini saya mengganti id user dari guest ke admin di kolom pencarian





Dan ini Adalah tampilan dari user admin dan saya berhasil mendapatkan flag nya saat saya berhasil mengganti id nya dari guest ke admin
